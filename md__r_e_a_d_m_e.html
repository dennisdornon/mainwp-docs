README

Welcome to the MainWP repository on GitHub. Here you can browse the source, look at open issues and keep track of development.

Not a Developer

If you are not a developer, please use the MainWP Community for questions or issues.

Developer

The MainWP WordPress management system consists of 2 core plugins, the MainWP Dashboard plugin which is containted in this repository. The second plugin is the MainWP Child plugin, which can be found in this repository.

The MainWP Dashboard is a stand-alone plugin that is installed on its own WordPress site; no other plugins should be installed with it, and it should not be installed on an active WordPress site with visitors.

The MainWP Child plugin currently has an . The Child plugin is installed on the WordPress sites that you want to control from the MainWP Dashboard.

Support

This repository is not suitable for support. Please don't use our issue tracker for support requests, but for core MainWP issues only. Support can take place through the appropriate channels:

  • If you have a problem, you may want to start with the Help Docs.
  • The MainWP Community which is available for all MainWP users.
  • You can also open a support ticket.
  • For customizations take a look at our list of MainWP Experts or Codeable.

Support requests in issues on this repository will be closed on sight.

Contributing to MainWP

If you have a patch or have stumbled upon an issue with the MainWP Plugin you can contribute this back to the code. Please read our contributor guidelines for more informationon how you can do this.
