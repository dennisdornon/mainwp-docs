Detailed Description

REST Controller

This class extend WP_REST_Controller in order to include /batch endpoint for almost all endpoints in MainWP REST API.

It's required to follow "Controller Classes" guide before extending this class: https://developer.wordpress.org/rest-api/extending-the-rest-api/controller-classes/

NOTE THAT ONLY CODE RELEVANT FOR MOST ENDPOINTS SHOULD BE INCLUDED INTO THIS CLASS.
