When you\textquotesingle{}re utilizing an SFTP stream you\textquotesingle{}re not calling the methods in this class directly -\/ PHP is calling them for you. Which kinda begs the question... what methods is PHP calling and what parameters is it passing to them? This function lets you figure that out.
