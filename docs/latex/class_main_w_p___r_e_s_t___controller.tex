\doxysubsection{Detailed Description}
REST Controller

This class extend {\ttfamily WP\+\_\+\+REST\+\_\+\+Controller} in order to include /batch endpoint for almost all endpoints in \doxylink{namespace_main_w_p}{Main\+WP} REST API.

It\textquotesingle{}s required to follow "{}\+Controller Classes"{} guide before extending this class\+: \href{https://developer.wordpress.org/rest-api/extending-the-rest-api/controller-classes/}{\texttt{ https\+://developer.\+wordpress.\+org/rest-\/api/extending-\/the-\/rest-\/api/controller-\/classes/}}

NOTE THAT ONLY CODE RELEVANT FOR MOST ENDPOINTS SHOULD BE INCLUDED INTO THIS CLASS.
