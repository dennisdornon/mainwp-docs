Returns the OID corresponding to a name

What\textquotesingle{}s returned in the associative array returned by load\+X509() (or load\texorpdfstring{$\ast$}{*}()) is either a name or an OID if no OID to name mapping is available. The problem with this is that what may be an unmapped OID in one version of phpseclib may not be unmapped in the next version, so apps that are looking at this OID may not be able to work from version to version.

This method will return the OID if a name is passed to it and if no mapping is avialable it\textquotesingle{}ll assume that what\textquotesingle{}s being passed to it already is an OID and return that instead. A few examples.
