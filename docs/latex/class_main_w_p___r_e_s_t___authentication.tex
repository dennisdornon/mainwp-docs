Main\+WP\+\_\+\+REST\+\_\+\+Authentication\+::get\+\_\+authorization\+\_\+header (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Get the authorization header.

On certain systems and configurations, the Authorization header will be stripped out by the server or PHP. Typically this is then used to generate {\ttfamily PHP\+\_\+\+AUTH\+\_\+\+USER}/{\ttfamily PHP\+\_\+\+AUTH\+\_\+\+PASS} but not passed on. We use {\ttfamily getallheaders} here to try and grab it out instead.
