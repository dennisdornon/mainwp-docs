The \doxylink{namespace_main_w_p}{Main\+WP} Word\+Press management system consists of 2 core plugins, the \doxylink{namespace_main_w_p}{Main\+WP} Dashboard plugin which is containted in this repository. The second plugin is the \doxylink{namespace_main_w_p}{Main\+WP} Child plugin, which can be found in \href{https://github.com/mainwp/mainwp-child}{\texttt{ this repository}}. ~\newline


The \href{https://wordpress.org/plugins/mainwp/}{\texttt{ Main\+WP Dashboard}} is a stand-\/alone plugin that is installed on its own Word\+Press site; no other plugins should be installed with it, and it should not be installed on an active Word\+Press site with visitors.
