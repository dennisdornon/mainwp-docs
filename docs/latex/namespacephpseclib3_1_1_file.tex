\doxysubsection{Detailed Description}
Pure-\/\+PHP ANSI Decoder

PHP version 5

If you call read() in \textbackslash{}phpseclib3\textbackslash{}\+Net\textbackslash{}\+SSH2 you may get {\ttfamily \doxylink{}{ANSI escape codes}} back. They\textquotesingle{}d look like chr(0x1\+B) . \textquotesingle{}\mbox{[}00m\textquotesingle{} or whatever (0x1B = ESC). They tell a {\ttfamily \doxylink{}{terminal emulator}} how to format the characters, what color to display them in, etc. \textbackslash{}phpseclib3\textbackslash{}\+File\textbackslash{}\+ANSI is a {\ttfamily \doxylink{}{VT100}} terminal emulator.
